# Upload to hc-pragma-ai Organization

## Files

- **avatar-icon.png** — Simple square icon (recommended for org avatar)
- **org-avatar.png** — Full logo version

## Steps

1. Go to: https://github.com/orgs/hc-pragma-ai/settings/profile
2. Under **Profile picture**, click **Upload new picture**
3. Select `avatar-icon.png` (or `org-avatar.png`)
4. Crop to square if prompted
5. Click **Save**

GitHub requires manual upload — no API available.
